Projects

    • Music Pins

    A music-oriented Pinterest clone (2013, capstone project at AppAcademy).
    • Users create pins by associating a picture with an artist and a song.
    • Pick a picture from your Facebook or Instagram account, or via a Flickr search.
    • Browse other pins relating to the same artist, song, or user.
    • Input song/band with help from the Freebase API
    • RESTful API, Responsive layout, Infinite scrolling.

    Tech used: Ruby on Rails, Backbone JS, Freebase API

    Try it / See on Github
